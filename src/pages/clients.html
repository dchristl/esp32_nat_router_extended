Add Static IP Assignment

Static IP Assignment ensures devices connecting to the ESP32 receive the same IP address each time they connect. To configure this, the following settings need to be chosen:
  • Static IP: 192.168.4.2
  • MAC Address: AA:BB:CC:DD:EE:FF
When the device with MAC Address "AA:BB:CC:DD:EE:FF" connects to the ESP32, it will always receive the IP Address "192.168.4.2".
